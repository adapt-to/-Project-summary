1. 预期目标
==============

本文档的预期目标是能够实现一套适用于双协议的诱骗态量子光源设备。具体的预期指标如下:

.. note::
    1. 实现双协议兼容并能够快速切换
    2. 实现软件部分控制硬件部分的快速控制
    3. 设备在随机数下发过程中能够长时间稳定性( *8小时以上* )
    4. 实现的双协议为BB84协议和COW协议，并且BB84协议传输频率为10Mhz，COW协议的传输频率为40Mhz
